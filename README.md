# LURE Framework
**Lifecycle of User-targeted Risk and Exploitation**

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19478053.svg)](https://doi.org/10.5281/zenodo.19478053)

> A structured kill-chain framework for mapping criminal pathways in consumer-facing cybercrime.

---

## Overview

The **LURE Framework** is an adversary behavior framework designed to systematically describe how threat actors target, manipulate, and exploit individual consumers through digital and social engineering channels.

Inspired by MITRE ATT&CK, LURE maps criminal operations across eight sequential stages — from infrastructure setup through exploitation and exit — and cross-references seven distinct harm categories that victims may experience.

---

## Structure

### Stages (S1–S8)

| ID | Stage | Description |
|----|-------|-------------|
| S1 | Resource Dev. | Acquisition of tools, accounts, and infrastructure |
| S2 | Target Selection | Reconnaissance and victim identification |
| S3 | Approach | Initial contact with the target |
| S4 | Trust Shaping | Building psychological trust and rapport |
| S5 | Behavioral Steering | Manipulating target behavior toward desired actions |
| S6 | Compromise | Seizing access, control, or leverage |
| S7 | Exploitation | Extracting value — financial, informational, or physical |
| S8 | Cover & Exit | Concealment of traces and preparation for the next cycle |

### Harm Categories (H1–H7)

| ID | Harm |
|----|------|
| H1 | Financial / Property |
| H2 | Information / Privacy |
| H3 | Sexual Exploitation |
| H4 | Psychological / Mental |
| H5 | Social / Reputational |
| H6 | Legal Entanglement |
| H7 | Physical / Life |

## Usage

Open either HTML file in a browser:

1. **Select a Harm** (H1–H7) to highlight the relevant criminal path
2. **Tap a Mode** to view detailed adversarial acts for that stage
3. Tap **×** to close the detail panel
