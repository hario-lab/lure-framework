# LURE Framework
**Lifecycle of User-targeted Risk and Exploitation**
**消費者サイバー犯罪における被害経路の構造化フレームワーク**

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19478053.svg)](https://doi.org/10.5281/zenodo.19478053)

> 個人を標的とするサイバー犯罪の攻撃経路を体系的に記述するためのキルチェーン型フレームワーク。

---

## 概要

**LURE Framework** は、脅威アクターが個人（消費者）をデジタルおよびソーシャルエンジニアリング手法によって標的化・操作・搾取する手口を体系的に記述することを目的とした、敵対的行動フレームワークです。

MITRE ATT&CK にインスパイアされ、LURE はインフラ準備から搾取・逃走に至る8つのステージにわたって犯罪行動をマッピングし、被害者が経験しうる7種類の被害カテゴリと照合します。

---

## フレームワーク構造

### ステージ（S1–S8）

| ID | ステージ | 説明 |
|----|---------|------|
| S1 | Resource Dev. / 準備 | ツール・アカウント・インフラの調達 |
| S2 | Target Selection / 標的選定 | 偵察および被害者の特定 |
| S3 | Approach / 接触 | 標的への初回コンタクト |
| S4 | Trust Shaping / 信頼形成 | 心理的な信頼・ラポートの構築 |
| S5 | Behavioral Steering / 行動誘導 | 望む行動へ向けた標的の誘導 |
| S6 | Compromise / 侵入・掌握 | アクセス・支配・弱みの掌握 |
| S7 | Exploitation / 搾取 | 金銭・情報・身体的価値の収奪 |
| S8 | Cover & Exit / 隠蔽・逃走 | 痕跡隠滅と次サイクルへの準備 |

### 被害カテゴリ（H1–H7）

| ID | 被害 |
|----|------|
| H1 | 金銭・財産 |
| H2 | 情報・プライバシー |
| H3 | 性的搾取 |
| H4 | 心理・精神 |
| H5 | 社会的信用 |
| H6 | 法的巻き込み |
| H7 | 身体・生命 |

## 使い方

HTMLファイルをブラウザで開いてください。

1. **Harmを選択**（H1〜H7）すると、該当する犯罪経路のModeが強調表示されます
2. **Modeをタップ**すると、そのステージの具体的な攻撃行動（Acts）が表示されます
3. **×ボタン**で詳細パネルを閉じます
